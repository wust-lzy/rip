#  RIP路由协议  基于Java Swing实现



- Message.java    定义路由表的数据结构
- NetWork.java    定义网络的数据结构
- Router.java         定义路由器的数据结构
- SubWindow.java   右边小窗口的swing程序
- Window.java   主窗口swing程序



在main函数中初始化路由网络拓扑图

运行main函数即可
